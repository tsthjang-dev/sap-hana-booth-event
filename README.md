# Event Booth App (5-Step Mission)

## Run
```bash
cd /Users/tsth/.openclaw/workspace/event-booth-app
npm install
npm start
```

- Participant page: http://localhost:4010
- Admin dashboard: http://localhost:4010/admin.html

## Features
- Team registration (1~3 members)
- Team name as key
- Sequential booth progress (1 -> 5)
- Simple answer check per booth
- Final feedback submit
- Admin progress overview

## Notes
- Data is saved in `data.json`
- Booth answers in `server.js` (`boothAnswers`)
